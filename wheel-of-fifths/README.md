# Wheel of Fifths (2026-06)

This model has a few moving parts:

- A base with 84 musical notes arranged in a circular grid.
  - Around the circle, the notes are arranged in Circle of Fifths order
  - Each column contains 5 notes (also in Circle of Fifths order)
- Several other discs with windows cut out
  - This allows highlighting certain relationships between the notes. E.g. the chords of a key, or relative modes.

This is intended as a music theory tool.
