# Drawing Templates (2026-06)

This model is a rectangular frame for tracing out rectangles on paper. This
allows me to quickly set up bounds for my art.

## Template Sizes

The template sizes are controled by the `SHAPE` variable near the top of the
file.

- `SHAPE = ATC;` - Artist Trading Card format (2.5x3.5 inches). This is a key
  - size for my art
- `SHAPE = BOOKMARK;` - 2x6 inch bookmark
